Gem Jam

A browser based game built using the Phaser.io framework and HTML5. Exploring a 2D platformer world, the player must locate gems while navigating dangerous pitfalls and traps. The game features a timer as well as a score to compare previous attempts. I used the application Tiled to help me in generation a large 2D side scrolling world using an open source, small-sized spritesheet to allow for quick loading times. The game uses states to keep the code organized and easy to use. All the assets are preloaded for quickness once the actual game has begun.

Accio Bike

Seattle has one of the highest bike theft rates in the nation with over 1,550 reported stolen last year alone. Many end up being sold or bartered as currency on the street. For a savvy thief, stealing a bike is easy. For a victim, getting a bike back is hard.

Accio Bike is a registry for your bicycle to proactively prevent and recover thefts. Without creating an account, a user can search the database of missing bikes for information like a bicycle's unique serial number, brand, model, color, size, or the zip code from which it went missing. If your bike has gone missing, create a free account and fill in your bike's information. The app data scrapes common sites to sell stolen goods and matches them to your missing bike's profile.

This was a group collaboration using Ruby on Rails for the web framework, PostgreSQL for the database, Nokogiri gem for data scraping, Cloudinary for image uploads, Rspec for testing, and Git for version control.
